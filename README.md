# 🧠 CPU Scheduling Algorithm Simulator

Simulator algoritma CPU Scheduling berbasis Python untuk membantu memahami cara kerja penjadwalan proses pada sistem operasi.

Project ini mengimplementasikan beberapa algoritma scheduling populer lengkap dengan perhitungan:

* Waiting Time
* Turnaround Time
* Completion Time
* Gantt Chart
* Average Waiting Time
* Average Turnaround Time

---

# 🚀 Algoritma yang Tersedia

Project ini mendukung 5 algoritma CPU Scheduling:

| Algoritma                            | Deskripsi                                        |
| ------------------------------------ | ------------------------------------------------ |
| FCFS (First Come First Serve)        | Proses dijalankan berdasarkan urutan kedatangan  |
| SJF (Shortest Job First)             | Proses dengan burst time tercepat diprioritaskan |
| Priority Scheduling                  | Prioritas tertinggi dijalankan terlebih dahulu   |
| Round Robin                          | Setiap proses mendapat jatah waktu (quantum)     |
| Shortest Remaining Time First (SRTF) | Versi preemptive dari SJF                        |
